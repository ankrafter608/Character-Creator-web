# Character Creator Web

Инструмент для создания персонажей и лорбуков для AI-ролеплея.

### ⚠️ Важное примечание
Для корректной работы **обязательно** нужен собственный пресет с настроенным **jailbreak**. Без него AI может отказываться генерировать контент или форматировать его в нужный JSON.

### Состояние проекта
Проект на данный момент **очень сырой и дерьмовый**. Пользоваться можно, функции свои выполняет, но удобство интерфейса и стабильность пока оставляют желать лучшего.

## Особенности
- **System Prompts**: Полный контроль над "мозгами" приложения. В настройках можно редактировать системные промпты для каждой страницы (генерация персонажа, лорбука, скрапера и т.д.).
- **Autonomous Mode**: Экспериментальная фича. Агент, который может сам гуглить википедию, читать страницы и наполнять лорбук/карточку. Работает нестабильно, может тупить, но в целом фурычит.
- **Wiki Scraper**: Умный загрузчик данных. AI сам подбирает нужные Fandom-вики и страницы.
- Редактирование карточек персонажей (SillyTavern V2).
- Создание и управление лорбуками.
- Встроенный AI-чат для помощи в генерации контента.
- Генератор промптов для артов (ComfyUI / Stable Diffusion).
- Кастомизируемый ресайз чата и шрифт JetBrains Mono.

## Установка и запуск
1. `npm install`
2. `npm run dev`
